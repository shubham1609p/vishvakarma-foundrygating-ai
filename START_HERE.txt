TROUBLESHOOTING

❌ Backend won't start: "Python not found"
   → Install Python from https://www.python.org/
   → Make sure to check "Add Python to PATH"
   → Restart your command prompt

❌ Error: "ModuleNotFoundError: No module named 'flask'"
   → Run: pip install flask google-generativeai python-dotenv flask-cors
   → Try again

❌ Browser can't connect to backend
   → Is the command prompt still showing "[START] ✅ Server ready!"?
   → If not, the server crashed - check error message
   → Try running with: python -u backend.py (for more output)

❌ Chat shows error: "AI service currently unavailable"
   → Backend is not running
   → Check Step 3 and 4 above
   → Restart backend

❌ Chat shows error: "Authentication failed"
   → API key issue (shouldn't happen with default key)
   → In production, set: GEMINI_API_KEY=your_real_key
